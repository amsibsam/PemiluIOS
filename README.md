#  Pantau Bersama
Sebuah gerakan aplikasi gotong royong 


## Requirements

XCode 10, iOS 10 SDK or later

## Runtime

iOS 10 or later

### Dependency Management

#### Cocoapods

> https://cocoapods.org

#### Bundler

> http://bundler.io

#### Fastlane

>https://docs.fastlane.tools/getting-started/ios/setup/

### How to Run

1. git clone 
1. cd PantauBersama
1. bundle install
1. bundle exec pod install
1. open PantauBersama.xcworkspace
